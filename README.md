# DeployForge

DeployForge is a microservices-based cloud deployment platform that automatically builds and deploys GitHub repositories as Docker containers.

## Architecture

Services included:

- API Gateway
- Orchestrator
- Docker Manager
- Log Service
- Frontend Dashboard

## Tech Stack

Backend: FastAPI (Python)  
Queue: Redis + Celery  
Containerization: Docker  
Frontend: React  
Database: MongoDB
